# My old dotfiles (from when I used Arch Linux)
- Dotfile: A file or directory whose name begins with a dot (period or full stop), typically hidden from view.
- User-specific application configuration is traditionally stored in so called dotfiles.


## Credits:
- The original waybar config: [Anik200/dotfiles](https://github.com/Anik200/dotfiles/tree/super-waybar).


## Software I used:
> Colour scheme: Gruvbox\
> Display manager: sddm\
> Window manager: Hyprland (wayland)\
> Bar: Waybar\
> Terminal: Kitty\
> Shell: zsh (with p10k)\
> Text editor: Neovim\
> App launcher: Rofi\
(I still use most of these lol)


## Fonts:
- JetBrains Mono 
- JetBrains Mono Nerd


### To download all dependencies:
1. `$ sudo pacman -S nvim hyprland hyprpaper waybar rofi fastfetch zsh ttf-jetbrains-mono ttf-jetbrains-mono-nerd`
2. Copy the fonts in `.config/waybar/fonts` to `/usr/local/share/fonts/{otf,ttc,ttf}/`
